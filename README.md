# Documentations YULPA avec VuePress

Documentations à destination des clients yulpa.

## Setup

### Pré-requis

1.  [Node.js](https://nodejs.org/en/) v8.0+

### Instructions

1.  Clone the repo
2.  Run `yarn` or `npm install`
3.  Run `yarn dev` or `npm run dev`
